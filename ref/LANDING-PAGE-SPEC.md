# Especificação da Landing Page: ProdAPS

## Objetivo
Converter Secretários e Coordenadores de Saúde demonstrando controle e transparência.

## Estrutura de Seções
1. **Hero**: 
    - Headline impactante sobre produtividade na APS.
    - Subheadline mencionando a substituição de planilhas manuais.
    - CTA: "Solicitar Acesso".
2. **Pain (O Caos das Planilhas)**:
    - Grid visual mostrando os problemas: "Dados perdidos", "Falta de metas", "Preenchimento difícil".
3. **Solution (O Painel de Controle)**:
    - Mockup do Dashboard com os Gauges (Velocímetros) coloridos.
    - Texto sobre a clareza visual (Vermelho ao Azul).
4. **Features Key**:
    - Mobile-first para Diretores.
    - Gestão de metas flexível.
    - Painel de pendências em tempo real.
5. **Social Proof/Trust**:
    - Frase sobre conformidade com a gestão de dados da Atenção Primária.
6. **Final CTA**:
    - Botão de destaque para contato/demo.

## Diretrizes de Layout
- **Visual**: Minimalista, fundo claro (#FAFAFA), muitas bordas arredondadas (shadcn style).
- **Imagens**: Screenshots reais do dashboard e do formulário mobile.
